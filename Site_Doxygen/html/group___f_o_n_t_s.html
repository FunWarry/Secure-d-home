Detailed Description

Fonts library for all my LCD libraries.

Default fonts library. It is used in all LCD based libraries.

Supported fonts

Currently, these fonts are supported:

  • 7 x 10 pixels
  • 11 x 18 pixels
  • 16 x 26 pixels
