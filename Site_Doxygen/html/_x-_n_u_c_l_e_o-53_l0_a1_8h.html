◆ VL53L0A1_GPIO1_C_OPTION

#define VL53L0A1_GPIO1_C_OPTION   0

select GPIO1 to exti mapping for center or shared interrupt

Set option value or un-define it to match with board configuration

  • not defined or 0 : U14=On and U17=off => GPIO1_C = PA4
  • defined and not 0 : U14=Off and U17=on => GPIO1_C = PC1
